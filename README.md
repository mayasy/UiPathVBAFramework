# UiPathVBAFramework
UiPath から [Execute Macro](https://docs.uipath.com/activities/lang-ja/docs/execute-macro) アクティビティを使用して呼ばれることを想定した VBA のフレームワーク。
Module1.bas は **標準モジュール** に対してインポートしないと、UiPath 側で返却値が取得できない点に注意して下さい。
